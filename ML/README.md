# MLOps(Machine Learning Operations)

MLOps는 엔터프라이즈 애플리케이션을 효율적으로 개발, 배포, 실행하는 프로세스나 기술인 DevOps를 기반으로 한 기술이다. MLOps는 데이터 세트를 준비하고 이를 분석하는 AI 모델을 만드는 데이터 사이언티스트 팀을 지원하며, 모델을 통해 체계적으로 자동 처리하는 머신러닝 엔지니어도 포함된다.
